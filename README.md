
## Installation

```st
Metacello new
	repository: 'github://michaelrauh/moldable-fold:main/src';
	baseline: 'MoldableFold';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfMoldableFold asClass loadLepiter
```
